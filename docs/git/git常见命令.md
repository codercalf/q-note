| ------------------------------------------------------------ | ------------------------------- |
| git init                                                     |                                 |

| git status / git status -s / git status --short              |                                 |
| git add                                                      |                                 |

| git commit -m                                                |                                 |
| git commit -a -m                                             |                                 |
| git rm                                                       | 此次修改次文件还没有提交        |
| git rm -f                                                    |                                 |
| git rm --cache                                               | 本地不删除                      |
| git mv <old file> <new file>                                 |                                 |
| 等同 mv <old file> <new file> , git rm <old file>, git add <new file> |                                 |

| git log -p / git log --patch -<number>                       |                                 |
